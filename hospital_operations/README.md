# أدوات إدارة بيانات المستشفيات والسيارات

هذا المجلد يحتوي على السكريبتات الأربعة المطلوبة لإدارة بيانات مقدمي الخدمة والسياسات التأمينية:

1. **`prepare_hospital_import.py`**:
   - يقوم بتحويل ملف الإكسيل القادم من المستشفى (بعد skipping 8 rows) إلى ملف جاهز للاستيراد.
   - يقوم بتصنيف الخدمات تلقائياً بناءً على التخصص والاسم.
   - الاستخدام: `python prepare_hospital_import.py "اسم الملف.xlsx"`

2. **`seed_coverage_rules.py`**:
   - يقوم بتوليد قواعد التغطية (حوالي 32 قاعدة) لسياسة تأمينية محددة.
   - الاستخدام: `python seed_coverage_rules.py <رقم_السياسة>`

3. **`clear_provider_contract.py`**:
   - يقوم بتفريغ عقد مقدم خدمة من جميع الخدمات المضافة مسبقاً.
   - الاستخدام: `python clear_provider_contract.py <رقم_العقد>`

4. **`convert_pdf_to_rules.py`**:
   - سكريبت لتحويل جداول المنافع المستخرجة من ملفات PDF إلى صيغة قواعد التغطية.
   - الاستخدام: `python convert_pdf_to_rules.py "ملف_الجدول.xlsx"`
